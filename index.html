Docker Webserver Projekt

Willkommen auf meinem Docker Webserver

Diese Webseite wird in einem Docker Container mit Apache betrieben. Der Container laeuft auf Port 8080 und ist jetzt im Browser erreichbar.

Dieses Projekt zeigt, wie Docker verwendet wird, um einen Webserver mit Apache schnell zu starten und zu betreiben.

Projektmerkmale

Docker Container

Die Anwendung laeuft in einem isolierten Docker Container, der einfach zu verwalten und zu skalieren ist.

Apache Webserver

Verwendet den Apache HTTP Server, um die Webseite zu hosten.
